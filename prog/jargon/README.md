INSTALLATION README
===================
This is the hacker jargon file reader.

Copy "jargon.l1com" to your "/prog" L1VM folder.
Copy "jargon" directory into your "home/user/l1vm" folder.

Now build the pov edit program:

```
$ ./build-l1vm.sh jargon
```

And run the program

```
$ l1vm prog/jargon
```

Have some fun!
